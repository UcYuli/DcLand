我應該選擇哪一種方式？

根據您的需求與業務類型，選擇最適合的管道。

全程網辦

  • ✓ 適合簡易、無財產爭議之申請。
  • ✓ 電腦 + 讀卡機 或 手機身分識別。
  • ✓ 免出門，審查完畢直接生效。

非全程網辦

  • ✓ 適合買賣、贈與等重大財產異動。
  • ✓ 線上上傳附件，地政事務所預審。
  • ✓ 僅需臨櫃核對身分，大幅縮減等待期。

常見問題

您只需在手機下載「行動自然人憑證」App 並完成註冊，登入數位櫃臺時選擇行動身分識別，再以手機掃描 QR Code 進行生物辨識（指紋或人臉）即可完成驗證，無須讀卡機。
是的。因為買賣或繼承涉及重大財產權移轉，法律規定必須由承辦人員當面核對身分與權狀正本。但線上先行申請與預審，能確保資料正確，將原本 1-2 小時的現場流程縮短為 10 分鐘左右。
