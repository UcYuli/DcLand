我應該選擇哪一種方式？

根據您的需求與業務類型，選擇最適合的管道。

全程網辦

  • ✓ 適合簡易、無財產爭議之申請。
  • ✓ 電腦 + 讀卡機 或 手機身分識別。
  • ✓ 免出門，審查完畢直接生效。

非全程網辦

  • ✓ 適合買賣、贈與等重大財產異動。
  • ✓ 線上上傳附件，地政事務所預審。
  • ✓ 僅需臨櫃核對身分，大幅縮減等待期。

常見問題

您只需在手機下載「行動自然人憑證」App 並完成註冊，登入數位櫃臺時選擇行動身分識別，再以手機掃描 QR Code 進行生物辨識（指紋或人臉）即可完成驗證，無須讀卡機。
是的。因為買賣或繼承涉及重大財產權移轉，法律規定必須由承辦人員當面核對身分與權狀正本。但線上先行申請與預審，能確保資料正確，將現場流程縮短。
數位櫃臺提供 24 小時線上登入查詢服務。但若是案件提交，建議在上班時間操作以利後續即時處理。

您可以透過手機下載「行動自然人憑證」並完成綁定，之後只需透過手機掃碼與生物辨識即可，完全不需要讀卡機。
申請人必須是權利人本人，並使用其憑證登入。如果是代理申辦，代理人需有相關證明文件並符合代理申報規定。
這是數位櫃臺提供的一項輔助證明，可用於向銀行或查驗單位展示產權狀態，具備時效性與防偽標章，與傳統紙本權狀具有對應資訊。
若地政事務所審核後發現資料不足，會透過系統通知您。您只需再次登入數位櫃臺，上傳缺漏的附件電子檔即可，不需跑現場。
